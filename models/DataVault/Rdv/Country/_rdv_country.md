{% docs rdv_country_h %}
This table is part of the Raw Data Vault (RDV) layer and serves as the country Hash table. It stores the hash keys for countries along with their original IDs, load timestamps, and record sources.
{% enddocs %}

{% docs rdv_country_h_hk_country_h %}
Hash key for the country. This is a unique identifier generated for each country record.
{% enddocs %}

{% docs rdv_country_h_country_id %}
The original country ID from the source system.
{% enddocs %}
